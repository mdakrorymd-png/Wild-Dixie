# رفع Wild Dixie Escapes على دومين مجاني (Render)

الطريقة دي بتديك **رابط ثابت مجاني** (زي `wild-dixie-web.onrender.com`) + قاعدة بيانات مجانية — من غير ما تدفع حاجة.

## المطلوب منك (مرة واحدة)
1. حساب **GitHub** مجاني → ارفع مجلد المشروع `egypt-rentals` كـ repository.
2. حساب **Render** مجاني (سجّل بحساب GitHub) → https://render.com

## الخطوات
1. ادخل Render → **New +** → **Blueprint**.
2. اختار الـ repo بتاع المشروع → Render هيقرأ ملف `render.yaml` ويعمل:
   - قاعدة بيانات PostgreSQL مجانية
   - سيرفر الباك-إند (FastAPI) — بيعمل migrations ويزرع بيانات تجريبية تلقائيًا
   - سيرفر الفرونت-إند (Next.js)
3. اضغط **Apply** واستنى البناء يخلّص.
4. بعد ما الباك-إند يشتغل، انسخ رابطه (مثلاً `https://wild-dixie-api.onrender.com`) وحطّه في إعدادات الفرونت (`wild-dixie-web` → Environment):
   - `BACKEND_ORIGIN` = `https://wild-dixie-api.onrender.com`
   - `API_INTERNAL_URL` = `https://wild-dixie-api.onrender.com/api/v1`
   - `NEXT_PUBLIC_SITE_URL` = رابط الفرونت (مثلاً `https://wild-dixie-web.onrender.com`)
   - (اختياري) `NEXT_PUBLIC_FB_PIXEL_ID` = الـ Pixel بتاعك للإعلانات
   ثم **Save** → الفرونت هيعيد البناء.
5. افتح رابط الفرونت — ده دومينك المجاني الجاهز للإعلانات.

## ملاحظات مهمة
- **النسخة المجانية بتنام** بعد ١٥ دقيقة خمول، وأول زيارة بعدها بتاخد ~٣٠–٥٠ ثانية تصحى. كفاية للتجربة؛ للإطلاق الجدّي رقّيها لخطة مدفوعة رخيصة.
- **قاعدة البيانات المجانية في Render بتتمسح بعد ٣٠ يوم** — للتجربة تمام، للإنتاج خطة مدفوعة.
- **استيراد Airbnb** (Playwright/Chromium) مش هيشتغل على النسخة المجانية (محتاج متصفح تقيل) — باقي الموقع كله شغّال.
- **دومين حقيقي (.com):** الـ `.onrender.com` مجاني تمامًا. لو عايز اسم خاص زي `wilddixie.com` ده بيتشترى (~١٠$ بالسنة من Namecheap) وبتربطه بـ Render في دقيقتين — قولّي أوريك إزاي.

## بديل سريع للتجربة دلوقتي (من غير أي حساب)
ملف `mobile.cmd` بيطلّعلك رابط عام مؤقت فورًا (cloudflared) — كويس للتجربة على الموبايل، بس بيتغيّر كل تشغيل. للإعلانات استخدم رابط Render الثابت.
